# Examples

Complete example apps built on the rss.chat APIs. Each one lives in its own folder with its own readme -- background, instructions, and everything needed to run it.

* [firehose](firehose/) -- two apps, one for Node and one for the browser, that listen to a server's firehose and show each post as it arrives.
* [threadwalker](threadwalker/) -- walks a whole conversation from the RSS feeds alone, no API calls, and prints it as an indented outline.
* [wordpress](wordpress/) -- rss.chat meets WordPress. The first app, inboundRss, mirrors posts from an rss.chat feed onto a WordPress site as they happen.
